## 🎯 Как запустить

### 1. Сборка клиента
```bash
cd client
npm run build
```

### 2. Запуск клиента
```bash
# В папке client
npm start

# Или из корневой папки
npm run client
```

### 3. Запуск сервера (в другом терминале)
```bash
# В папке server
npm start

# Или из корневой папки
npm run server
```

## 🔧 Что происходит при запуске

1. **npm start** автоматически:
   - Собирает React приложение (`npm run build`)
   - Запускает Electron с собранными файлами

2. **Electron загружает**:
   - `dist/index.html` - если папка существует
   - Показывает ошибку - если папка не найдена

3. **Меню окна**:
   - Полностью отключено
   - Нет File, Edit, View, Window меню

## 📁 Структура файлов

```
client/
├── dist/           # Собранные файлы (создается после build)
│   ├── index.html
│   └── bundle.js
├── src/            # Исходный код
│   ├── components/ # React компоненты
│   ├── contexts/   # React контексты
│   └── styles/     # CSS стили
├── main.js         # Electron main
├── preload.js      # Electron preload
└── package.json
```

## 🚨 Возможные проблемы

### Ошибка "dist папка не найдена"
```bash
cd client
npm run build
npm start
```

### Ошибка зависимостей
```bash
cd client
npm install
npm run build
npm start
```

### Ошибка портов
- Клиент: 3001 (Webpack dev server)
- Сервер: 3000 (Express)
- Electron: загружает из dist папки
